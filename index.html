Metode Analisis Terlengkap

Didukung oleh algoritma optimasi terbaru untuk akurasi maksimal

⏱️

Cooper-Jacob

Metode straight-line untuk data jangka panjang dengan koreksi konstanta presisi tinggi

  • Validasi kondisi u < 0.05
  • R² linearitas check
  • Confidence intervals
📐

Theis

Curve matching dengan optimasi non-linear gradient descent

  • Well function presisi
  • Residual minimization
  • Multiple starting points
🔄

Recovery Test

Analisis data residual drawdown untuk verifikasi parameter

  • Slope analysis
  • Transmisivity verification
  • Skin factor estimation
📊

Step Drawdown

Analisis efisiensi sumur dengan persamaan non-linear

  • Well loss coefficient (C)
  • Aquifer loss (B)
  • Efisiensi calculation
💧

Hantush

Leaky aquifer analysis dengan modified Bessel function

  • Leakage factor (B)
  • Vertical K estimation
  • Aquitard properties
⛰️

Neuman

Unconfined aquifer dengan delayed yield

  • Early & late time analysis
  • Specific yield (Sy)
  • Delay index
